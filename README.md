<h2>서비스 기획 의도</h2>
온라인 만남들이 다시 오프라인 만남으로 제자리를 찾아가는 위드코로나 시기이다.
개발자들도 지역 근처 다양한 개발자들과 만나 모각코 모임을 갖고 교류할 수 있는 플랫폼이 필요하다고 생각했다.

`GatherCoding` 은 사용자들이 클릭만으로 모각코 모임을 만들고, 참여할 수 있는 플랫폼을 목표로 한다.
